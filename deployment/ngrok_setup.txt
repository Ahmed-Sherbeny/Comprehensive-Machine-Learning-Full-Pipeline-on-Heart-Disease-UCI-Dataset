Make a config file for streamlit in the following directory:
    C:\Users\user_name\.streamlit

Config contents:
    [server]
    headless = true
    enableCORS = false
    enableXsrfProtection = true        
    port = 8501

    [browser]
    gatherUsageStats = false

Add your authtoken from https://dashboard.ngrok.com/get-started/setup/windows

Then run the following commands in your terminal:
    ngrok config add-authtoken your_auth
    ngrok http 80      # or your desired port number

Then run from your app terminal:
    streamlit run your_path_to\app.py
